The data files for this project are not included in this repository due to their large size.

## Source

The dataset can be downloaded from the official Kaggle competition page:

CMI - Detect Behavior with Sensor Data
https://www.kaggle.com/competitions/cmi-detect-behavior-with-sensor-data/overview

Instructions

1.  Go to the Kaggle link above and download the data files.
2.  Unzip the downloaded files (if necessary).
3.  Place the required data files directly into this `/data` directory.

The main script expects to find the following files in this folder:

train.csv
test.csv
train_demographics.csv
test_demographics.csv
sample_submission.csv
kaggle_evaluation/
